Angular Material components

Below are the default Angular Material components provided at https://material.angular.io/components/categories. I copied the code provided in the example components exactly with almost no modifications (I changed some colors and some of the text content and I removed the text inputs from the Dialog example). The components are built using Angular with Angular Material added and then built for web.

These components were choosen to see how accessibly the default components are rendered to the web.

Feel free to do you own accessibility testing on these components.
