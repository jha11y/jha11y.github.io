React Native components built for web

Below are the default React Native components provided at https://reactnative.dev/docs/components-and-apis. I copied the code provided in the example components exactly with almost no modifications (I added a second example with an image to test to see if alt text would show up because I was curious). The components are built using Expo CLI to be displayed on the web using React Native for web (this is done automatically by Expo).

These components were choosen to see how accessibly the default components are rendered to the web instead of iOS and Android.

Feel free to do you own accessibility testing on these components.

  • Button
  • FlatList
  • Image
  • Modal
  • Switch
  • TextInput
  • VirtualizedList
